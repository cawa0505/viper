In case of read operations

```
$ vm ls
host/vm_name
```

```
$ vm ps
host/vm_name
```

In case of create:

```
Usage: vm create <image> [/path/to/provision_script.sh] [--host=<host>]
```

In case of control commands:

```
$ vm start [host/]vm_name
```
```
$ vm stop [host/]vm_name
```
```
$ vm ssh [host/]vm_name [options]
```
